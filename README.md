# GraphQL Demo 
Code example from GraphQL

## Quick start

```bash
# clone the repo
git clone https://github.com/pedrowss/graphqldemo.git

# change into the repo directory
cd graphqldemo

# install
npm install

# run json server
$ npm run json:server

# run 
$ npm run dev
```

Then visit [http://localhost:4000/graphql](http://localhost:4000/graphql) in your browser.
